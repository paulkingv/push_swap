# push_swap
PUSH_SWAP is a project where we must choose and implement an algorithm to sort a set of numbers into ascending order in the fewest amount of steps.\

We are allowed the use of two *stacks*, called **stack_a** and **stack_b**, to use to sort our set of numbers. A stack, in my implementation, will be a linked list.

## Description
### Rules
Push_Swap subject permits us to use the following ####Operations\
**Swap**
`sa`    Swap the two top elements of stack_a\
`sb`    Swap the two top elements of stack_b\
`ss`    Exec `sa` and `sb` at the same time\
**Push**
`pa`    Take the first element of stack_b and put it at the top of stack_a (Does nothing if stack_b is empty)\
`pb`    Take the first element of stack_a and put it at the top of stack_b (Does nothing if stack_a is empty)\
**Rotate**
`ra`    Shift up all elements of stack_a by 1\
`rb`    Shift up all elements of stack_b by 1\
`rr`    Exec `ra` and `rb` at the same time\
**Reverse Rotate**
`rra`   Reverse rotate stack_a (Shifts all elements in the stack down one. Bottom becomes the top)\
`rrb`   Reverse rotate stack_b (Shifts all elements in the stack down one. Bottom becomes the top)\
`rrr`   Exec `rra` and `rrb` at the same time
